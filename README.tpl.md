# weixin-fe

2022 年前端圈微信公众号合集

## 定义

#### 分类

聚焦 FE 角度分类。

纯手打收集，不保证分类、名称等信息一定正确。

#### 活跃

**2022 年** 有发文。

#### 未录入

非 FE / 文章少 / 领域涉猎广 / 无法定性 等情况不录入。

近期有改名的、久远未发文等情况的未做录入。

#### 排序

尊重分享，优先展示活跃部分，同类排序无先后顺序。
